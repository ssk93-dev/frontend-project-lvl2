## About
Compares two configuration files and shows a difference.<br>
### Example for 2 JSON files
```sh
{
    "host": "hexlet.io",
    "timeout": 50,
    "proxy": "123.234.53.22",
    "follow": false
  }
```
```sh
{
    "timeout": 20,
    "verbose": true,
    "host": "hexlet.io"
  }
```
